# gymnote
ВАЖНО: ПЕРЕД ЗАПУСКОМ НАДО ЗАПУСТИТЬ JSON-SERVER --WATCH DB.JSON и потом уже программу.

Программа в ветке master. На первой странице выбираем язык программы и переходим на страницу авторизации. 
Вводим в форму авторизации данные и отравляем при помощт fetch (адрес и т.д. еще не заполнено). Если данные почты и пароля уже существуют и совпадают с базой, входим в аккаунт, если нет то повторяем заново. (Сдесь пока логика не дописана т.к. пока не знаю какуб инфу ты мне вернешь с сервера).
При успешном входе в аккаут перемещаемся на страницу ввода упражнений. Логика здесь такова: после заполенения поля "упражнение" и снятии фокуса идет запрос на сервер с данным названием упражнения и сервер возвращает массив с данными по форме как в файле db.json. Сейчас логика получения офрмлена через RTK Query в автоматическом режиме, но когда ты сделаешь базу данных хочу переписать логику получения данных по упражнениям также через RTK Query, но уже как пангинация данных (чтобы не грузить лишней информации с сервера).
При нажатии на кнопку "Сохранить" собранные данные отправляются на сервер для записи. Пока запись идет в файл db.json
